# Smart Irrigation System

IoT irrigation monitoring and control system.

## Project Structure

```
smart-irrigation-system
├── backend   -> Spring Boot API
└── frontend  -> React Native Expo App
```

## Features

- Monitor temperature
- Monitor soil moisture
- Pump ON/OFF control
- Automatic mode
- MySQL database
- Mobile application

## Technologies

### Backend
- Java 17
- Spring Boot
- MySQL
- JPA/Hibernate

### Frontend
- React Native
- Expo
- TypeScript
